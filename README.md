# Relatorio Meta Ads: Figueira Marketing

Landing page estatica feita pela Figueira Marketing com relatorio comparativo de campanhas Meta Ads para imoveis na faixa de R$ 600 mil.

## Como abrir

Abra `index.html` no navegador. O botao `PDF / Imprimir` permite salvar o relatorio como PDF.

## Arquivos

- `index.html`: estrutura e conteudo do relatorio
- `styles.css`: layout responsivo e estilos de impressao
- `script.js`: acionamento do botao de impressao
- `assets/figueira-marketing-logo.png`: logo usado no topo do relatorio
